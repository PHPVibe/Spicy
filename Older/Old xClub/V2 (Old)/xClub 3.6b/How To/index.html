Description

With PHPVibe RedTube Importer you can import videos from redtube based on tags, searches, channel, star

Installation procedure

Prepare upload

Unpack the plugin to a folder on your hard drive.

Backup your site!

Copy the entire content of the folder "plugin" to your PHPVibe's root on server. Exclude the folder "plugin" just it's content.

Chmod

The permissions of folder "moderator/redtube" and the files inside it set to 0777

If you have not done any changes to your site's files : /lib/class.providers.php you can safely and fast install this plugin by overwriting the files on server. and that's all...

If you have done changes to /lib/class.providers.php follow this:

For /lib/class.providers.php user a text comparing software (there are plenty free) to check the differences.

That's all.

You should see now two links under the menu "Video importers" (expand this first).
